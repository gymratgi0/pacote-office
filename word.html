Programa do pacote Office: WORD

O que é o Word?

O Microsoft Word é uma das maiores plataformas para desenvolver arquivos e foi pensado para criar documentos profissionais e de qualidade, para que o seus projetos sejam otimizados e organizados de maneira eficaz. A edição de texto é algo que faz parte do cotidiano da maioria das pessoas, seja para redigir um trabalho ou relatório, fazer anotações ou mesmo para elaborar um business plan para a sua empresa. Por isso, a Microsoft criou uma ferramenta para explorar todas possibilidades de processamento de texto: o Word. Vale ressaltar que processar um texto vai muito além de apenas editar os documentos. Fatores como design, elementos gráficos e formatação também são considerados nesse processo.

Atalhos no Word

Atalhos Descrição
CTRL + A Abre um arquivo já existente
CTRL + B Salva o documento
CTRL + E Centraliza o texto
CTRL + I Formata o texto para itálico
CTRL + L Abre a pesquisa
ALT + CTRL + F Insere uma nota de rodapé
ALT + CTRL + I Entra no modo de visualização de impressão
CTRL + ENTER Começa uma nova página no mesmo documento
CTRL + U Substitui texto, formatação ou outros itens
CTRL + X Recorta o texto selecionado
